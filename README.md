# SO_Depth_Genetics
Multi-taxon synthesis of COI barcoding data investigating depth-driven population structure and cryptic species diversity in the Southern Ocean — shelf (0–500 m) vs. bathyal (500–3,000 m) | R pipeline | Biodiversity and Conservation 

This repository contains the complete reproducible analysis pipeline for a multi-taxon COI barcoding synthesis investigating whether depth (shelf 0–500 m vs. bathyal 500–3,000 m) structures genetic diversity and species boundaries in Southern Ocean marine fauna. Using 2,642 publicly available sequences from NCBI GenBank, we show that depth operates at two biological scales simultaneously: as a population-level filter in high-dispersal nototheniid fish (Dissostichus eleginoides, Lepidonotothen squamifrons) and as a hard species boundary in low-dispersal invertebrates (Harmothoe polychaetes: 8 MOTUs; Eusirus amphipods: cryptic species complex). The pipeline covers automated sequence download, ETOPO1 bathymetric depth recovery, MAFFT alignment, species delimitation, AMOVA, admixture analysis, PCA, Mantel IBD tests, and publication-ready figure generation.
